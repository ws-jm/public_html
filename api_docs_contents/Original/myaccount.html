7.   MyAccount

My Account

This section allows you to retrieve your account information from the server, request a new API key or request the account password be resent (to the users registered email address).

There are 4 account endpoints:

  • GetAccountDetails – Returns detailed information about your user account.
  • RequestNewAPIKey – Returns a new API key from the server.
  • RequestAPIKey – Request a new API key to be emailed to your registered address.
  • SendPasswordReset – Request to reset your password using an emailed link.
