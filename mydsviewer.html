Filter:
You can filter results using words and/or phrases (in double quotes).
Filters are not case sensitive and apply to the Symbol, Category and Description columns only.
Double quotes must be preceded by the escape "\" character.
Words or phrases can be excluded with a preceding "-" minus sign.

For example, a search using: "Crude Oil" -Brent
Returns series containing the phrase "Crude oil" but not the word "Brent".
